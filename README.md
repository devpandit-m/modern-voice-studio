# 🎙️ Modern AI Voice Studio

A sleek, high-performance **Text-to-Speech (TTS)** web application that converts text into natural-sounding audio. This project focuses on a premium user experience using modern CSS techniques and the Web Speech API.

![License](https://img.shields.io)

## ✨ Features

- **Crystal Clear Audio:** Leverages the `SpeechSynthesisUtterance` API for high-quality voice output.
- **Voice Customization:** Dropdown menu to select from various system voices (Male/Female/Global Accents).
- **Speed Control:** Adjustable playback rate (0.5x to 2x).
- **Glassmorphism UI:** A beautiful "frosted glass" effect using `backdrop-filter`.
- **Neon Animation:** A dynamic, 5-color rotating neon border for a futuristic feel.
- **Fully Responsive:** Optimized for both Desktop and Mobile browsers.

## 🚀 Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Advanced animations, Conic-Gradients, and Backdrop-blur.
- **JavaScript:** DOM Manipulation and Web Speech API.
